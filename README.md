# Hooks
Third party integrations to chatwoot

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'hooks'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install hooks
```
